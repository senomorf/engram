Engram.

Notes and voice, written into the photo itself.

Engram tucks a note or a short voice clip inside your photos and videos, so the memory travels wherever the file goes. No account, no cloud.

Download the APK View on GitHub
Android 13+ - free and open source
✈

Works offline

Writing, browsing, search, verify and export all run with no network.

🔒

Private by design

No accounts, no telemetry. Nothing leaves your phone unless you share the file.

🌐

English and Russian

Full bilingual UI with an in-app language switch. Voice dictation in both.

📷

The file is the memory

Notes ride inside the JPEG, PNG or MP4 as versioned, checksummed records.

One honest limit

Embedded memories survive file-copy paths (backups, drives, original-quality cloud) and are stripped by recompressing share paths, like most messengers. Share the file itself to keep the memory attached, and use the built-in verifier to check any backup.
